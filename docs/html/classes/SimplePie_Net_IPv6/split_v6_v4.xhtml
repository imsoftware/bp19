SimplePie_Net_IPv6::split_v6_v4

Splits an IPv6 address into the IPv6 and IPv4 representation parts

RFC 4291 allows you to represent the last two parts of an IPv6 address using the standard IPv4 representation Example: 0:0:0:0:0:0:13.1.68.3 0:0:0:0:0:FFFF:129.144.52.38
