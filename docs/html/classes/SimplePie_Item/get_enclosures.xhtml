SimplePie_Item::get_enclosures

Get all available enclosures (podcasts, etc.)

Supports the <enclosure> RSS tag, as well as Media RSS and iTunes RSS. At this point, we're pretty much assuming that all enclosures for an item are the same content. Anything else is too complicated to properly support.
