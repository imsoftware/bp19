SimplePie::subscribe_url

Get the URL for the feed

When the 'permanent' mode is enabled, returns the original feed URL, except in the case of an `HTTP 301 Moved Permanently` status response, in which case the location of the first redirection is returned. When the 'permanent' mode is disabled (default), may or may not be different from the URL passed to , depending on whether auto-discovery was used.
